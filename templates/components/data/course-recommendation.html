Course Recommendation

dataset of paper: Hierarchical Reinforcement Learning for Course Recommendation in MOOCs
We collect the dataset from XuetangX——one of the largest MOOCs in China and the dataset is available. (data.csv)
Data Description:
stu_id: student id.
time: the first time that the student enrolled the corresponding course.
course_index: the course id.
name: the name of the course.
type: the type of the course.
type_id: the type id.
